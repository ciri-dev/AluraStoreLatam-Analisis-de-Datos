# AluraStoreLatam-Analisis-de-Datos
Este repositorio contiene un análisis exploratorio de datos (EDA) sobre las ventas de distintas tiendas de Alura Store, utilizando Python y librerías de ciencia de datos.
El objetivo principal es obtener insights comerciales, identificar patrones de compra y visualizar la distribución geográfica de las ventas.


## Contenido del proyecto

 - Carga y limpieza de datos desde archivos CSV
 - Análisis de ingresos por tienda
 - Análisis de categorías y productos
 - Visualización de datos con gráficos estadísticos
 - Mapa de calor geográfico basado en ciudades donde hay mas ventas
 - Informe final y conclusiones.
